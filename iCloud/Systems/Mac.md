# Keys

| Keys             | Action                          |
| ---------------- | ------------------------------- |
| Ctrl  ->         | move between desktops           |
| Ctrl  1-9        | Switch to desktop number        |
| Ctrl  Alt 1-6    | Switch to desktop numbers 11-16 |
| Ctrl Shift z/x/c | Move window to screen           |
| Cmd Ctrl <-      | left half of screen             |
| Cmd Ctrl m       | Maximise                        |
