
# Data quality

- Consistency: data is collected and recorded in the same way.
- Accuracy: error-free
- Uniqueness: no duplicate entries
- Validity: data is fit for intended use. e.g. dates in the right format
- Timeliness: is the data relevant to the time it was collected
- Completeness: how much of total data is collected

Careful Analysts usually verifies timely completeness

# Uncertainty

Aleatoric - inherent that is always there like flipping a coin. Estimating house prices because buyers vary.
Epistemic - uncertainty about a system. If you only did a ML on 10 houses then the model has uncertainty.

All the knowledge would still have Aleatoric uncertainty.
