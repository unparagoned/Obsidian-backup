Power BI is a collection of software services, apps and connectors that work together to turn data in visually immersive and interactive insights.

Methods

Desktop

Service allows you to share reports, can have real time data ingestion

Mobile mainly just for viewing reports

Queries are different sheets

Analytics record is used for analysis.

Two main engines,

- Power query, cleaning merging, etc.
- Power pivot, data analytics, relationships between tables and hierarchical structures

Both are available in excel

Power Query is data transformation and preparation. Used for ETL

Has GUI and formula language called M.

Power pivot is in memory modelling component that has highly compressed data storage and extremely fast aggregation and calculation

No limit on number of rows/columns

Power Pivot language called DAX(Data Analysis eXpressions)

Power Query(connect/marge..) -> Power Pivot(setup relationships, calculate extra columns, etc..) -> Power BI(Visualize)
