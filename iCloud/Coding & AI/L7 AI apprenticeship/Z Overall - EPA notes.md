# Professional discussion

Can have journal with you. Can refer to this as much as you want.

60min

10 open questions

Easy to pass and easy to fail. - IF journal is in a good state then it's easy but otherwise it can be hard.

Say what PCA is, don't need the maths, how have you used it in your work. If you haven't applied it say that but give theoretical situation of when and why you would use it.

Summarise all the assignments done

Performance metrics - it always comes up

e.g.  Why is precision better than accuracy for cancer detection

Define compare contrast

ML algorithms

List all the PAI ML algorithms, then, for each algorithm, define; and state how you either used that algorithm or you would use it in the workplace. Try to stick to the STARR method where applicable 

Summarize the requirements for all the assignments you have completed and briefly describe what you did for each -- aim for 10 lines on each assignment. The reason behind this is that you have everything in one place in case a question arises about one of the assignments during the professional discussion. 

List the evaluation metrics you learned with a few use cases. Focus on why certain evaluation metrics are more important than others in a given scenario (for example, why precision is more important than accuracy in cancerous cells detection).

BCS L7 Exam Prep Guide

DELTA Framework

**D – Define**

**E – Explain**

**L – Link / Logic**

**T – Trade‑offs / Tests / Tables (context‑dependent)**

**A – Answer / Action / Apply**

e.g. if there are some outliers you might choose k-media over k-means - tradeoff

Make sure you have a conclusion.
